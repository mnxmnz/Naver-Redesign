## Getting Started

#### Prerequisites

```
npm install --global yarn
```

#### Installation

1. Clone the Repo

```sh
git clone https://github.com/mnxmnz/Naver-Redesign.git
```

```sh
cd Naver-Redesign
```

2. Install Project Packages

```sh
yarn
```

3. Run the Project (Dev Mode)

```sh
yarn start

http://localhost:3000/
```
